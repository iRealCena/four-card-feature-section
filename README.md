# Four Card Feature Section

A fully responsive four card feature section built with semantic HTML and modern CSS techniques.

## 🚀 Overview

This project is a responsive feature section layout that showcases four different feature cards in a structured grid.  
The layout adapts seamlessly across desktop, tablet, and mobile devices.

## 🛠️ Built With

- HTML5 (Semantic Markup)
- CSS3
- CSS Grid
- Flexbox
- CSS Variables
- clamp() for fluid typography
- Media Queries for responsiveness

## 📱 Responsive Design

- Desktop → CSS Grid layout
- Tablet → Adjusted Grid positioning
- Mobile → Flex column layout
- Conditional line break handling for better text flow

## 🎯 Features

- Clean and modern UI
- Responsive grid system
- Reusable CSS variables
- Fluid typography using `clamp()`
- Optimized spacing and alignment
- Mobile-first adjustments

## 🧠 What I Practiced

- Responsive layout restructuring
- Managing breakpoints effectively
- Using clamp() for scalable font sizes
- Clean CSS architecture
